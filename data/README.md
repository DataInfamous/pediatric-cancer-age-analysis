# Dataset

This folder contains the dataset used for the pediatric cancer age incidence analysis.

## Source

The data comes from the **Pediatric Cancer Knowledgebase (PeCan)**.

PeCan is an integrated resource developed by St. Jude Children's Research Hospital that provides genomic and clinical data on pediatric cancers.

PeCan Portal  
https://pecan.stjude.cloud

## File

PeCan_sample_info_20260312.xlsx

This file contains clinical metadata for pediatric tumor samples including:

- diagnosis_subtype_name
- diagnosis_subtype_code
- age_at_diagnosis_yrs
- subtype_biomarkers
